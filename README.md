# Michelle Tran · 陳氏清花 — Portfolio

Personal portfolio website of **Michelle Tran (陳氏清花)** — International Marketing & Sales · AI Product Practitioner. NCKU IMBA. Bridging Vietnam and international markets.

A single-page, bilingual (**English ⇄ 繁體中文**) site with a storytelling journey timeline, built as a static `index.html` (no build step).

## View locally

Just open `index.html` in any browser — no server or dependencies needed.

## Deploy (GitHub Pages)

In this repo: **Settings → Pages → Source: `main` branch / root** → the site goes live at
`https://michelle-tw.github.io/portfolio/`.

## Sections

- **Hero** — intro, positioning, key stats
- **My Story** — narrative + language proficiency
- **The Journey** — chronological timeline (Vietnam → Taiwan → entrepreneurship → career → global competitions → building with AI)
- **Expertise & Recognition** — skills, AI & tech, education, certifications
- **Video Showcase** — embedded pitch/project videos
- **Contact** — email, LinkedIn, GitHub, 104
